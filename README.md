### инициализируем наш GO проект

```
$ go mod init go_*
```

### команда, go mod tidy, добавить отсутствующие и удалить неиспользуемые модули 

```
$ go mod tidy
```

### запуск
```
# запустить
$ go run .

# создайте приложение с помощью команды
go build -o go_*

# посмотреть версию GO
go version
```
